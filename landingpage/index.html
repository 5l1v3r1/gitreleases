Git Releases allows you to directly link to the assets of your latest release on GitHub.

Enter a GitHub repo

Available asset links

  • gitreleases.dev/gh/rokka-io/rokka-go/latest/rokka_macos.zip
  • gitreleases.dev/gh/rokka-io/rokka-go/latest/rokka_windows.zip

URL format

https://gitreleases.dev/gh/{owner}/{repo}/{tag}/{assetname}

{tag} can be a real tag or "latest".
